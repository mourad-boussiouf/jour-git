Commandes Usage
git init Initialise un répertoire Git.
git add . Ajouter tous les fichiers d'un dossier au répertoire get d'un coup.
git add nom_du_fichier Ajouter un fichier au répertoire git.
git commit Commit les fichiers ajoutés au répertoire git.
