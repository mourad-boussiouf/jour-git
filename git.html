Commandes Usage
git init Initialise un répertoire Git.
git add . Ajouter tous les fichiers d'un dossier au répertoire get d'un coup.
git add nom_du_fichier Ajouter un fichier au répertoire git.
git commit Commit les fichiers ajoutés au répertoire git.
git checkout SHA Permet d'avoir les informations sur un commit grâce à son identifiant SHA
git checkout -b branch_name Permet de créer une branche et d'aller dessus.
git checkout branch_name Permet d'accéder à une branche éxistante.
git merge branch_name Permet d'intégrer les modifications faites sur branche.
git log Permet d'avoir la liste des commits effectués avec leurs commentaires
git status Permet de savoir sur quelle branche nous sommes ainsi que les fichiers ajoutés actuellement
git clone URL Permet de cloner localement un répertoire disponible sur Github
git remote add origin URL Relie le répertoire local au répertoire en ligne Github spécifié.
git push origin branch_name Pousse les mofifications locales vers le répertoire Github
git pull origin branch_name Transfère les fichiers Github vers notre répertoire local.
